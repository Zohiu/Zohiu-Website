About Me

Hello! I'm Zohiu, a programmer and designer from Germany. Most of the projects I work on are private, but there are some things I can show. I also produce music, which you can find out more about by scrolling down.

If you like my stuff, I'm open for commissions.

Projects

Stuff I published

  • dotfiles: My desktop config for Arch Linux.

Public projects

  • re[Parse]: A small development team working on a Markdown-based editor.
  • Juky: A pretty configurable audio visualizer using neopixels to be put between an audio source and an amplifier - Currently still unfinished
  • Minecraft Tools: I count this as one project - This is where I put all the stuff I do related to Minecraft, including some Plugins.
  • Wave Machine: Creates wave patterns in water using an Arduino, a fish tank and two paddles.
  • Sinewave Demo: Related to the Wave Machine. Uses math to manipulate a sinewave, so a servo could be controlled by it.
  • File To Picture: Convert any binary file to a PNG and back.
  • Godot Raycaster: Not very advanced, but shows the basics of how raycasting can be used to render a 2.5D world.
  • Python Raycaster: Same as the above, but a bit older. Just a demo.

Discontinued projects

  • Nota: My first ever project - A note-taking bot for Discord. I discontinued it once slash-commands started being enforced.
  • Spotify Audio Visualizer: Mainly intended for use with a Beamer and a fog machine to create some nice light breams. I switched to something else, so I no longer need my own app for this.
  • Bettermarks Cheater: My old school used Bettermarks to track homework, but I didn't wanna do it so I automated the point-gathering to get a good grade. I no longer go to that school, so the project has been archived.

There are many other projects that aren't public at the moment, mostly due to them containing personal data or raising potential legal/copyright issues.
